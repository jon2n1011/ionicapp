Sobre la aplicación

Nuestra aplicación esta destinada para aquella gente que quiere tener una vida saludable y relacionada con el deporte. Con ella podrás crear todo tipo de rutinas a tu gusto, cocinar las recetas que más te gusten de nuestra aplicación y muchas cosas más.

Preguntas frecuentes

1) ¿Cómo accedo a las rutinas?

Para acceder a las rutinas tenemos que pulsar cualquiera de las imagenes que van pasando del primer carrusel de imagenes, una vez dentro se nos mostrarán todas las rutinas que tenemos creadas.

2) ¿Cómo creo una rutina?

Para poder crear una rutina tenemos que estar dentro de "Crea/Modifica tu rutina", una vez dentro le damos al botón más para poder crear nuestra rutina con los ejercicios.

3) ¿Cómo accedo a las recetas?

Para acceder a las recetas tenemos que pulsar cualquiera de las imagenes que van pasando del segundo carrusel de imagenes, una vez dentro se nos mostrará una lista de recetas que podemos seleccionar.

4) ¿Puedo modificar mis datos?

Sí, solo tienes que ir a la opcion de "Perfil" del menú lateral y una vez dentro darle al lápiz para editar tus datos.

5) Quiero saber mi peso ideal

Para saber el peso ideal tienes que seleccionar la calculadora que esta en el menú lateral, una vez dentro introduce tus datos para calcular tu peso ideal.
